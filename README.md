# Blockchains Common Cross-Language Translations

See [AGENTS.md](AGENTS.md) for a complete description.

**Note:** This is a *preview* and may require changes for the orchestration system to run in any given environment. In particular it may have dependencies expressed as symbolic links to directories outside this repo's file system.

## Status - Preview

These libraries are not ready for usage in deployed services. They have been generated by agents, and the code has not been thoroughly reviewed for all languages, and it has not been adjusted for the idioms of each language. These tasks require human supervision, which is not currently funded for this work.

## Financial Support

If you would like to make use of one or more of these translations, [contact us](mailto:team@blockchaincommons.com) and we can talk to you about the financial support needed to do so.
